# TODO List 구현

## 프로젝트 시작 🚀

- 패키지 다운로드

```
npm i
# or
npm i --legacy-peer-deps
```

- 개발서버 시작

```
npm run start
```

개발서버 접근 URL : [http://localhost:3000](http://localhost:3000)

- Build

```
npm run build
```
